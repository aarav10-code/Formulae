# 📘 SSC Class 10 Maths & Science Formula Book

A simple and responsive HTML website containing important **Maharashtra SSC Class 10 Mathematics and Science formulae**. This project is designed for quick revision and can be hosted easily using **GitHub Pages**.

---

## ✨ Features

- 📚 Important Maths Formulae
- 📐 Geometry Formulae
- ⚡ Physics Formulae
- 🧪 Chemistry Formulae
- 📱 Responsive Design
- 🎨 Simple and Clean Interface
- 🌐 GitHub Pages Ready

## 🚀 Getting Started

1. Download or clone this repository.

```
git clone https://github.com/your-username/ssc-formula-book.git
```

2. Open **index.html** in any web browser.

---

## 🌍 Publish on GitHub Pages

1. Upload the project to GitHub.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, select:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
4. Save the settings.
5. Your website will be available at:

```
https://your-username.github.io/ssc-formula-book/
```
